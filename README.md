# Menufy

Menufy is an online menu system built to practice and evolve full-stack development using **Java, Spring Boot, and React**.

## Features
- Digital menu
- Product categories
- Admin panel (CRUD)
- Responsive UI

## Tech Stack
**Backend**
- Java
- Spring Boot
- Spring Data JPA
- PostgreSQL

**Frontend**
- React
- Vite
- Tailwind CSS

## Project Structure
- `/backend` → REST API
- `/frontend` → Web application
